# Atividade Landing Page

Um projeto simples de landing page para aplicar os conceitos que foi passando na aula de HTML.

Está incluso também o desafio de criar um calendário usando a tag <table>.

## TAGS usadas
    * <header>
    * <nav>
    * <ul>
    * <li>
    * <section>
    * <div>
    * <span>
    * <footer>
